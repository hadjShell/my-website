## Technique

### Chest

### Back

---

#### 宽距高位下拉

- 针对大圆肌，主上下回旋，无肩屈肩伸
- 拉到眼睛即可
- 手向桡偏，全握，大拇指食指发力

#### 反手高位下拉

- 针对背阔，主肩屈肩伸
- 身体微前倾，拉到腹肌能保持躯干稳定，肩不弯起来的的极限
- 手向尺偏，半握，后三指发力

#### 坐姿窄距划船

- 吸气吸满至背部有发力的感觉再启动
- 上肢重心前倾
- 大腿后侧蹬住发力

::: tip Self tip

注意颈椎段中立位

:::

::: info 针对背阔外延

- 含胸，想象展背的感觉
- 开肘

:::

::: info 整个背

- 挺胸，不要顶腰
- 夹肘
- **肩胛主动后缩**带着肩，不是肩带着肩胛骨

:::

#### 直臂下压

- 宽距，1.5 倍肩宽
- 扣腕抬肘

::: tip

这里的抬肘指的是伸直手臂自然抬起，全程保持

:::

::: info 针对大圆

- 俯身
- 只做前 2/3 行程

:::

::: info 针对背阔

- 几乎不俯身
- 针对后 2/3 行程
- 胸要挺做对抗

:::
