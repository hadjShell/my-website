## 蛙泳

::: important

呼吸是节奏的关键，呼吸正确身体就不累。

:::

**Pull-Kick-Glide**

1. 分手；

2. 先抬头，自然高肘划水至大臂收紧肋骨两侧，小臂手掌向上贴近身体，**口吸气**，脚仍然并紧，略微分开；

::: tip

抬头只需从向下看至向前看幅度，不要过大动作。
吸气注意防止假呼吸，把气“喝”进去，不要只张嘴。

:::

3. 先前送手，实际体感应为向前拉伸背阔肌，制造后侧链张力，为收腿做准备。头埋水成流线型，**入水憋气**；

4. 与上一步同时，收腿蓄力，主收小腿，脚掌向外翻，大腿夹角不宜过大；

5. 流线型建立后向后稍斜向外踢腿，**踢腿发力时呼气**，脚并紧；

::: tip

心态不要着急，感受脚外翻，身体建立发力预设，再呼气踢腿。

:::

6. 流线型滑行，**持续呼气**。
